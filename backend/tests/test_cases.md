# 初中数学残局挑战系统 - 测试用例文档

## 1. 测试用例概述

本文档包含了初中数学各阶段的解题例子，用于测试系统的AI辅助解题功能、苏格拉底式提示生成功能以及WebSocket通信功能。

## 2. 测试用例列表

### 2.1 初一数学测试用例

#### 2.1.1 有理数运算
**题目**：计算：(-3) × 4 + (-2)² - 15 ÷ 3  
**预期结果**：-13  
**AI提问**：你确定运算顺序正确吗？先算乘方，再算乘除，最后算加减，是这样吗？

#### 2.1.2 一元一次方程
**题目**：解方程：3(x - 2) + 1 = x - (2x - 1)  
**预期结果**：x = 3/2  
**AI提问**：去括号时，你注意符号变化了吗？特别是右边括号前面的减号，去掉括号后里面的每一项都要变号哦。

#### 2.1.3 几何初步（角的计算）
**题目**：已知∠AOB = 90°，OC是∠AOB的平分线，OD是∠BOC的平分线，求∠AOD的度数。  
**预期结果**：67.5°  
**AI提问**：你能画出图形吗？先画∠AOB=90°，再画平分线OC，最后画OD，这样会更清楚各个角的关系。

### 2.2 初二数学测试用例

#### 2.2.1 二元一次方程组
**题目**：解方程组：
```
2x + y = 8
x - y = 1
```  
**预期结果**：x=3，y=2  
**AI提问**：除了加减消元法，你还可以用代入消元法解这个方程组吗？试试看，哪种方法更简便？

#### 2.2.2 三角形全等
**题目**：已知AB=CD，BC=DA，求证：∠B=∠D  
**预期结果**：∠B=∠D  
**AI提问**：你确定用SSS判定全等吗？检查一下三个边是否都对应相等。另外，连接AC后，两个三角形的对应边是否正确？

#### 2.2.3 分式方程
**题目**：解方程：1/(x-2) + 3 = (x-1)/(x-2)  
**预期结果**：无解（增根x=2）  
**AI提问**：解分式方程一定要检验哦！你检验了吗？当x=2时，分母为0，所以x=2是增根，原方程无解。

### 2.3 初三数学测试用例

#### 2.3.1 一元二次方程
**题目**：用配方法解方程：x² + 6x - 7 = 0  
**预期结果**：x₁=1，x₂=-7  
**AI提问**：配方时，你知道要加上一次项系数一半的平方吗？一次项系数是6，一半是3，平方是9，所以要两边都加9，对吗？

#### 2.3.2 一次函数
**题目**：已知一次函数的图像经过点A(1, 3)和B(-1, -1)，求这个一次函数的解析式。  
**预期结果**：y=2x+1  
**AI提问**：你确定设的是一次函数的一般形式吗？一次函数的标准形式是y=kx+b，没错。那代入点坐标时，x和y的值有没有对应错？

#### 2.3.3 二次函数
**题目**：已知二次函数y = ax² + bx + c的图像经过点(0, 3)，(1, 4)，(2, 3)，求这个二次函数的解析式。  
**预期结果**：y=-x²+2x+3  
**AI提问**：你观察到点(0,3)和(2,3)的y值相同了吗？这说明什么？二次函数图像关于对称轴对称，对吗？

#### 2.3.4 圆的性质
**题目**：已知AB是⊙O的直径，C是⊙O上一点，OD⊥AC于D，若OD=3，求BC的长。  
**预期结果**：6  
**AI提问**：你记得垂径定理吗？垂直于弦的直径平分弦，所以D是AC中点。那OD和BC有什么关系呢？OD是△ABC的中位线，对吗？

#### 2.3.5 相似三角形
**题目**：在△ABC中，DE∥BC，AD=2，DB=3，DE=4，求BC的长。  
**预期结果**：10  
**AI提问**：你确定相似比是AD:AB吗？相似三角形的对应边要对应正确，AD对应AB，DE对应BC，对吗？

#### 2.3.6 三角函数
**题目**：在Rt△ABC中，∠C=90°，sinA=3/5，BC=6，求AB和AC的长。  
**预期结果**：AB=10，AC=8  
**AI提问**：你确定sinA是对边比斜边吗？∠A的对边是BC，斜边是AB，没错。那勾股定理应用正确吗？

## 3. 测试用例使用说明

1. **AI服务测试**：将上述测试用例用于测试AI服务的解题能力和提示生成能力
2. **苏格拉底式提示测试**：将测试用例中的AI提问用于验证系统的提示生成逻辑
3. **WebSocket测试**：使用这些测试用例模拟客户端发送的解题步骤，测试WebSocket通信功能
4. **完整性测试**：确保每个测试用例都能通过系统的处理流程，得到正确的结果

## 4. 测试执行要求

1. 所有测试用例必须通过自动化测试
2. 测试结果必须与预期结果一致
3. AI生成的提示必须符合苏格拉底式提问的要求，引导学生思考而非直接给出答案
4. WebSocket通信必须稳定，能够正确处理各种消息类型
